# Submission Checklist

## PromptWars portal

- Add the public GitHub repository URL
- Add the deployed app URL
- Add the LinkedIn post URL
- Confirm the selected challenge is `Physical Event Experience`

## Repository checks

- Public repo
- Single branch only
- Source-only repo
- README present
- No secrets committed

## Technical verification

- `npm run lint`
- `npm run test:unit`
- `npm run build`
- Open the live deployment and test all four scenarios

## Demo sanity checks

- Pre-entry Rush works
- Halftime Surge works
- Accessible Reroute works
- Emergency Exit Coordination works
- `Ask PulsePath AI` still degrades gracefully without a key

## Narrative assets

- Technical blog draft reviewed
- LinkedIn post published with setup photo or screen capture
- Use `#BuildwithAI #PromptWarsVirtual`
- Tag `@googlefordevelopers @hack2skill`

## Final reminder

If App Hosting or Gemini keys are not available, the app still remains demoable because the deterministic routing engine is client-visible and scenario switching can be tested locally.
